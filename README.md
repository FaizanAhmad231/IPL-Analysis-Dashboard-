# IPL Analysis Dashboard 🏏📊

An interactive Power BI dashboard for analyzing the Indian Premier League (IPL) from 2008 to 2025. This project provides detailed insights into team performance, player statistics, match records, season summaries, and tournament trends using dynamic visualizations and filters.

The dashboard is designed for cricket fans, analysts, and data enthusiasts who want to explore IPL data in a simple, professional, and visually appealing way.

---

## ✨ Features

- 📅 Season-wise IPL analysis
- 🏆 Champion & Runner-up details
- 📊 Interactive Points Table
- 🟠 Orange Cap statistics
- 🟣 Purple Cap statistics
- 🔥 Total Sixes & Fours analysis
- 🏟 Venue-wise insights
- 👥 Team & Player performance analysis
- 🎯 Dynamic filters and slicers
- 🎨 Professional dashboard UI

---

## 🛠 Tech Stack

- Power BI
- DAX
- Power Query
- Excel / CSV Dataset

---

## 📊 Dashboard Insights

- Highest run scorers
- Leading wicket takers
- Best performing IPL teams
- Team standings and trends
- Boundary statistics
- Match and venue analysis

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Refresh the dataset if required
4. Use slicers and filters to explore IPL statistics
